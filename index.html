This game was made as a hobby project to showcase some simple Javascript compentencies. It includes:

  • A simple tab-based inteface.
  • Built-in functions such as Math.floor() and Math.random().
  • Custom functions for loading data and showing a modal onload and dynamically changing the HTML.
  • Lists of objects encapsulating the monsters, player, and combat data.
  • Lists for handling monsters and combat rounds.
  • A custom implemented upgrade system for stats as well as equipment.

The full source can be found at my Github repository.
